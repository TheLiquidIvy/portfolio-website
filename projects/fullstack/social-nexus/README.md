# Social Nexus

A fully functional social media platform with real-time features, user authentication, and interactive messaging.

## Features

- **User Authentication**: Complete login/signup system with localStorage persistence
- **Post Creation**: Create posts with text and emoji support
- **Like System**: Interactive like functionality on all posts
- **Real-time Messaging**: Simulated real-time chat with multiple conversations
- **User Profiles**: View statistics including post count and total likes received
- **Feed System**: Dynamic feed displaying all user posts with timestamps

## Tech Stack

- **Frontend**: Vanilla JavaScript, HTML5, CSS3
- **Styling**: Cyberpunk theme with neon colors (#00ffff, #ff00ff, #00ff41)
- **Fonts**: Orbitron, Rajdhani
- **Storage**: localStorage for data persistence

## How to Use

1. **Sign Up**: Create a new account with username, email, and password
2. **Login**: Access your account with credentials
3. **Create Posts**: Share your thoughts with emoji support
4. **Interact**: Like posts from other users
5. **Message**: Chat with pre-populated demo users
6. **Profile**: View your stats and activity
