Task Matrix

A kanban-style task system that supports drag-and-drop planning, priority scoring, and team collaboration workflows with clean, responsive UI patterns.

React Redux DnD Kit CSS Modules

Project Goals

  • Make planning workflows intuitive and fast.
  • Provide keyboard-friendly task controls.
  • Support team assignments and status filters.

Key Features

  • Drag-and-drop kanban lanes and swimlanes.
  • Realtime progress overview widgets.
  • Saved views for personal and team boards.

Challenges Solved

  • Prevented layout jank during drag operations.
  • Created state slices for large board data.
  • Designed consistent card hierarchy at scale.
