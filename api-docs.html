Public JSON API for Portfolio Data

Overview

This API provides public access to portfolio data in JSON format. All endpoints are static JSON files with no rate limits.

Base URL: https://theliquidivyfile.github.io/portfolio-website/api/

Version: 1.0.0

Format: JSON

Authentication: None required

üìä Complete Portfolio Data

GET /api/portfolio.json

Returns complete portfolio information including contact details, skills, stats, and certifications.

Response Fields:
name - Full name
title - Professional title
about - Summary and location info
contact - Email, phone, social links
skills - Array of skills with levels
stats - Portfolio statistics
certifications - List of certifications
Example Request:
fetch('https://theliquidivyfile.github.io/portfolio-website/api/portfolio.json')
  .then(res => res.json())
  .then(data => console.log(data));

üìù Blog Posts

GET /api/blog.json

Returns all blog posts with metadata and statistics.

Response Fields:
blog_posts - Array of blog post objects
total - Total number of posts
categories - Available categories
total_views - Combined view count
total_likes - Combined like count

üí° Skills

GET /api/skills.json

Returns all technical skills organized by category with proficiency levels.

Response Fields:
skills - Array of skill categories
summary - Overall skill statistics
strengths - Key strengths
currently_learning - Skills in progress

üìà Statistics

GET /api/stats.json

Returns comprehensive portfolio statistics and activity data.

Response Fields:
overview - High-level statistics
github - GitHub activity and contributions
blog - Blog engagement metrics
experience - Professional experience data
activity - Recent activity and contribution patterns
